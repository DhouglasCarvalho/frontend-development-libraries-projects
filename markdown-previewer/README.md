# Markdown Previewer

A React-based Markdown previewer built for the freeCodeCamp Front End Development Libraries certification.

## Features

- Live Markdown editor
- Real-time HTML preview
- Supports GitHub-flavored Markdown
- Renders headings, links, code, blockquotes, lists, images, and bold text

## Technologies

- HTML
- CSS
- JavaScript
- React 17
- Marked.js
