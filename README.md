# Front End Development Libraries Projects

A collection of React projects built for the freeCodeCamp Front End Development Libraries certification.

## Projects

| Project | Description | Live Demo |
|---|---|---|
| Random Quote Machine | Generates random quotes and supports tweeting the current quote | https://codepen.io/dhouglascarvalho/pen/emBpqmw |
| Markdown Previewer | Converts Markdown into a live HTML preview | https://codepen.io/dhouglascarvalho/pen/ByQoXNr |
| Drum Machine | Interactive drum pad machine with click and keyboard controls | https://codepen.io/dhouglascarvalho/pen/ByQoXjx|
| JavaScript Calculator | Calculator with arithmetic operations, decimals, and chained expressions | https://codepen.io/dhouglascarvalho/pen/emBpqzO|
| 25 + 5 Clock | Pomodoro-style session and break timer | https://codepen.io/dhouglascarvalho/pen/KwNdOgE |

## Technologies

- HTML
- CSS
- JavaScript
- React 17
- Marked.js
- freeCodeCamp test suite

## Purpose

These projects demonstrate front-end component architecture, state management, event handling, controlled inputs, conditional rendering, timers, audio playback, and dynamic UI behavior.
