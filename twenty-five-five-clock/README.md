# 25 + 5 Clock

A React-based Pomodoro-style timer built for the freeCodeCamp Front End Development Libraries certification.

## Features

- Adjustable session length
- Adjustable break length
- Start, pause, resume, and reset controls
- Automatic session/break switching
- Countdown displayed in mm:ss format
- Audio alert when timer reaches zero

## Technologies

- HTML
- CSS
- JavaScript
- React 17
