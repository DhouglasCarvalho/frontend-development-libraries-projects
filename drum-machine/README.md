# Drum Machine

A React-based drum machine built for the freeCodeCamp Front End Development Libraries certification.

## Features

- Nine clickable drum pads
- Keyboard controls using Q, W, E, A, S, D, Z, X, and C
- Audio playback for each pad
- Display updates with the active sound name
- Responsive beat-pad style interface

## Technologies

- HTML
- CSS
- JavaScript
- React 17
