# Random Quote Machine

A React-based random quote generator built for the freeCodeCamp Front End Development Libraries certification.

## Features

- Displays a random quote on initial load
- Generates a new quote when the user clicks the New Quote button
- Displays the quote author
- Allows the user to tweet the current quote
- Responsive centered layout

## Technologies

- HTML
- CSS
- JavaScript
- React 17
