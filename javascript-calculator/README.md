# JavaScript Calculator

A React-based calculator built for the freeCodeCamp Front End Development Libraries certification.

## Features

- Addition, subtraction, multiplication, and division
- Decimal calculations
- Chained operations
- Clear/reset functionality
- Handles consecutive operators
- Prevents invalid multiple decimals
- Displays results with reasonable precision

## Technologies

- HTML
- CSS
- JavaScript
- React 17
